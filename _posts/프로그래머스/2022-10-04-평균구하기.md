## 📔 문제 설명

정수를 담고 있는 배열 arr의 평균값을 return하는 함수, solution을 완성해보세요.

💡 입출력 예

| arr       | result |
| --------- | ------ |
| [1,2,3,4] | 2.5    |
| [5,5]     | 5      |

## 💻내가 작성한 코드

```js
function solution(arr) {
  var answer = 0;
  let sum = 0;
  for (let i = 0; i < arr.length; i++) {
    sum += arr[i];
  }
  return sum / arr.length;
}
```

```js
function solution(array) {
  return array.reduce((a, b) => a + b) / array.length;
}
```
