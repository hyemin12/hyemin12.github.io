## 📔 문제 설명

정수 배열 arr가 주어집니다. arr의 각 원소에 대해 값이 50보다 크거나 같은 짝수라면 2로 나누고, 50보다 작은 홀수라면 2를 곱하고 다시 1을 더합니다.

이러한 작업을 x번 반복한 결과인 배열을 arr(x)라고 표현했을 때, arr(x) = arr(x + 1)인 x가 항상 존재합니다. 이러한 x 중 가장 작은 값을 return 하는 solution 함수를 완성해 주세요.

단, 두 배열에 대한 "="는 두 배열의 크기가 서로 같으며, 같은 인덱스의 원소가 각각 서로 같음을 의미합니다.

💡 입출력 예

| arr                    | result |
| ---------------------- | ------ |
| [1, 2, 3, 100, 99, 98] | 5      |

## 💻내가 작성한 코드

```js
function solution(arr) {
  var answer = 0;
  let prevArr = arr;
  while (true) {
    const nextArr = prevArr.map((n) => {
      if (n >= 50 && n % 2 === 0) {
        return n / 2;
      }
      if (n < 50 && n % 2 === 1) {
        return n * 2 + 1;
      }
      return n;
    });

    const isSameArray = prevArr.filter((n, i) => n !== nextArr[i]).length <= 0;
    if (isSameArray) break;

    prevArr = nextArr;
    answer += 1;
  }
  return answer;
}
```

## 💻다른 사람 코드

- every() : 모든 요소가 주어진 판별 함수를 통과하는지 테스트
- while문 조건으로 현재 배열, 이전 배열을 비교

```js
function solution(arr) {
  var answer = 0;
  let before = [-1];

  while (!arr.every((e, idx) => e == before[idx])) {
    before = [...arr];

    arr = arr.map((e) => {
      if ((e >= 50) & (e % 2 == 0)) return e / 2;
      if ((e < 50) & (e % 2 != 0)) return e * 2 + 1;
      return e;
    });
    answer++;
  }
  return answer - 1;
}
```
