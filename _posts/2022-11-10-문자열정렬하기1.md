## 📔 문제 설명

문자열 my_string이 매개변수로 주어질 때, my_string 안에 있는 숫자만 골라 오름차순 정렬한 리스트를 return 하도록 solution 함수를 작성해보세요.

💡 입출력 예
| my_string | result |
| ----------- | --------------- |
| "hi12392" | [1, 2, 2, 3, 9] |
| "p2o4i8gj2" | [2, 2, 4, 8] |
| "abcde0" | [0] |

## 💻내가 작성한 코드

```js
function solution(my_string) {
  return my_string
    .match(/[0-9]/g)
    .map((a) => a * 1)
    .sort((a, b) => a - b);
}
```
