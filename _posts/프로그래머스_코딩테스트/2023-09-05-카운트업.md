## 📔 문제 설명

정수 start_num와 end_num가 주어질 때, start_num부터 end_num까지의 숫자를 차례로 담은 리스트를 return하도록 solution 함수를 완성해주세요.

💡 입출력 예

| start_num | end_num |          result           |
| :-------: | :-----: | :-----------------------: |
|     3     |   10    | [3, 4, 5, 6, 7, 8, 9, 10] |

## 💻내가 작성한 코드

```js
function solution(flo) {
  return parseInt(flo);
}
```
