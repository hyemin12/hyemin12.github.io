## 📔 문제 설명

문자열 str이 주어질 때, str을 출력하는 코드를 작성해 보세요.

💡 입출력 예

`HelloWorld`

## 💻내가 작성한 코드

```js
const readline = require("readline");
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout
});

let input = [];

rl.on("line", function (line) {
  input = [line];
}).on("close", function () {
  str = input[0];
  console.log(str);
});
```
