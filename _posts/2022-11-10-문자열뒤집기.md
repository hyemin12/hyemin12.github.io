## 📔 문제 설명

문자열 my_string이 매개변수로 주어집니다. my_string을 거꾸로 뒤집은 문자열을 return하도록 solution 함수를 완성해주세요.

💡 입출력 예

| my_string | result  |
| --------- | ------- |
| "jaron"   | "noraj" |
| "bread"   | "daerb" |

## 💻내가 작성한 코드

```js
function solution(my_string) {
  return my_string.split("").reverse().join("");
}
```
