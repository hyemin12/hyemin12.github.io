## 📔 문제 설명

실수 flo가 매개 변수로 주어질 때, flo의 정수 부분을 return하도록 solution 함수를 완성해주세요.

💡 입출력 예

|  flo  | result |
| :---: | :----: |
| 1.42  |   1    |
| 69.32 |   69   |

## 💻내가 작성한 코드

```js
function solution(flo) {
  return parseInt(flo);
}
```
