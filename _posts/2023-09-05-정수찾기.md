## 📔 문제 설명

정수 리스트 num_list와 찾으려는 정수 n이 주어질 때, num_list안에 n이 있으면 1을 없으면 0을 return하도록 solution 함수를 완성해주세요.

💡 입출력 예

|      num_list       |  n  | result |
| :-----------------: | :-: | :----: |
|   [1, 2, 3, 4, 5]   |  3  |   1    |
| [15, 98, 23, 2, 15] | 20  |   0    |

## 💻내가 작성한 코드

```js
function solution(num_list, n) {
  return num_list.includes(n) ? 1 : 0;
}
```
