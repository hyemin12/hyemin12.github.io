## 📔 문제 설명

정수 n이 주어질 때, n이하의 짝수를 모두 더한 값을 return 하도록 solution 함수를 작성해주세요.

💡 입출력 예

| n   | result |
| --- | ------ |
| 10  | 30     |
| 4   | 6      |

## 💻내가 작성한 코드

```js
function solution(n) {
  var answer = 0;
  for (let i = 0; i <= n; i++) {
    if (i % 2 === 0) answer += i;
  }

  return answer;
}
```
