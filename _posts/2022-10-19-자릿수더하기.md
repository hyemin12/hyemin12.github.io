## 📔 문제 설명

자연수 N이 주어지면, N의 각 자릿수의 합을 구해서 return 하는 solution 함수를 만들어 주세요.
예를들어 N = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.

💡 입출력 예

| N   | result |
| --- | ------ |
| 123 | 6      |
| 987 | 24     |

## 💻내가 작성한 코드

```js
function solution(n) {
  return parseInt(
    String(n)
      .split("")
      .reduce((acc, cur) => Number(acc) + Number(cur))
  );
}
```
