## 📔 문제 설명

정수 n이 주어질 때, n을 문자열로 변환하여 return하도록 solution 함수를 완성해주세요.

💡 입출력 예

|  n   | result |
| :--: | :----: |
| 123  | "123"  |
| 2573 | "2573" |

## 💻내가 작성한 코드

```js
function solution(n) {
  return String(n);
}
```

```js
function solution(n) {
  return n.toString();
}
```
