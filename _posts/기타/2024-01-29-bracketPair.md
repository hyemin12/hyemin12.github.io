## VS Code 내장 기능 - bracket pair colorization

강사님이 VS Code extensions들 중 괄호의 열고 닫음을 보기 편하게 도와주는 `bracket pair colorization`을 추천하셨다.

그러나 밑줄이 쳐져있었고, 아래 설명을 읽어보니 VS Code에 기본 제공되기 때문에 사용되지 않는다고 적혀있었다.
